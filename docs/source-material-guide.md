# Source Material Guide

Use this folder for local reference material only.

Expected PDF path:

`source-material/pdf/PTG2E_MythicEchoes_Vol1.pdf`

Recommended working folders:

- `source-material/pdf/` - original PDF source.
- `source-material/art/` - local image crops or extracted artwork while building scenes/journals.
- `source-material/notes/` - local extraction notes, page maps, and scratch audits.
- `source-material/.cache/` - generated text, page images, OCR, and structured extraction output.

Do not commit source PDFs, full extracted book text, OCR caches, or copyrighted art. Build only the finished Foundry-native data that belongs in the distributable module.
