# Source Material

Place the source PDF here:

`source-material/pdf/PTG2E_MythicEchoes_Vol1.pdf`

This folder is for local working material only. The PDF, extraction caches, OCR output, image crops, and notes are intentionally ignored by git and excluded from release archives.

Suggested layout:

- `pdf/` - original PDF.
- `art/` - temporary art or map crops.
- `notes/` - local notes, page maps, and audits.
- `.cache/` - generated text, rendered page images, OCR, and structured extraction output.
